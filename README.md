# 全大運賽事醫護支援需知 v1.1.0

這是一個以 Next.js 14 + TypeScript + Tailwind CSS 製作的多頁互動課程網站。

## 功能
- 首頁與課程導覽
- 賽事總覽
- 角色分工
- 賽前準備檢核
- 紀錄與回報
- 後送與應變流程
- 單項競賽規則搜尋
- 場邊急症處理模組
- 情境測驗
- 重要窗口

## 啟動方式
```bash
npm install
npm run dev
```
